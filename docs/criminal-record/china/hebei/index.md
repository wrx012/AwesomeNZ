# 河北省 · 无犯罪记录证明

河北省居民可通过「河北省公安机关无犯罪记录证明」小程序在线申请无犯罪记录证明，无需现场办理。
## 办理渠道

**河北省公安机关无犯罪记录证明** 小程序 / App

- 微信搜索「河北省公安机关无犯罪记录证明」进入

## 申请前准备

- **本人有效身份证**
- **本人实名认证**
- **手机号**
